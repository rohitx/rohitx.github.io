Installation Guide to iPython Notebook

January 28, 2014

The installation of iPython notebook on a mac may seem a little more tricky than on Windows or Linux. In other OS, it is either double click or use apt-get install. However, for the mac, a sequeunce of steps need to be followed. Here, I highlight the steps that will help guide you to install iPython notebook on a mac.

I assume that you have the latest version of Python installed. Here are the steps to follow: First install pyzmq. For all of the installations you will need root access. So, install the first package with pip (or easy_install) in the following way: 

sudo pip install pyzmq
Next install Tornado. In my installation of Python, this was already installed. So, if you try to install it, you may get a message saying that it is already installed. Nonetheless, this step ensures that you have a version installed. Tornado is requires to support current version of browsers. 

pip install tornado

Next install jinja2. Ninja is a template tool for rendering HTML pages. 

pip install jinja2

That's it. Once these packages have been installed, you can test iPython Notebook installation with the following command in a terminal: 

ipython notebook

Searching for files like the FILE_SEARCH() function in IDL

January 05, 2014

One easy way to search for files is to make use of the python procedure glob.glob(). This function is very similar to IDL's file_search(). Here's how to use it: 

a = glob.glob("*/*.py")
type(a)
list

Printing out a will give you the list of files that match the search query. 

glob.glob("fullpath pattern")
